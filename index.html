关于hexo无法使用本地图片
发表于2021-01-25|更新于2021-01-28|Hexo|markdown调用图片•Hexo使用本地图片
原因 hexo中文章为markdown写法，当调用本地图片时出现无法调用或者一段时间后调用的图片发生变化导致图片加载不出来； 早期大部分的方案是把图片放在 source/img 下，然后在 markdown 里写 ![img] (/source/img/img.png) 。显然这样在本地的编辑器里完全不能正确识别图片的位置。 解决方案 在hexo主配置文件中将 post_asset_folder: false 改成 post_asset_folder:true ； 在hexo博客根目录执行cnpm install https://github.com/CodeFalling/hexo-asset-image –save； 执行hexo new “hello”，会在source/_posts里面生成hello.md文件与hello的文件夹； 把要添加的图片放在hello的文件夹里面，然后![title] (hello/图片名)来调用图片。
